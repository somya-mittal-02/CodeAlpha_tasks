# Task 1: DNA/Protein Sequence Analysis (HBB Protein)

This project focuses on the analysis of the Hemoglobin Beta (HBB) protein using Python-based bioinformatics approaches.

# Objective
To perform sequence analysis and identify homologous sequences using automated BLAST through Python.

# Methodology
- Retrieved HBB protein sequence from UniProt
- Used Biopython for sequence handling and analysis
- Performed BLAST analysis programmatically using NCBI BLAST
- Parsed BLAST results to extract similarity, identity, and alignment scores

# Results
- High sequence similarity observed across species
- Conserved regions identified indicating functional importance
- Results stored in XML format and analyzed using Python scripts

# Files Included
- `hbb.fasta` – protein sequence
- `hbb_filter.py`, `hbb_match.py`, `hbb_quality.py`, `hbb_similarity.py` – analysis scripts
- `hbb_result.xml` – BLAST output
- Screenshots and report included

# Conclusion
The Hemoglobin Beta protein shows strong evolutionary conservation. Python-based BLAST analysis provides an efficient and reproducible workflow for sequence analysis.
