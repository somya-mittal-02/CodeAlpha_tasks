# Task 2: Protein Structure Prediction (HBB Protein)

This task focuses on predicting and analyzing the 3D structure of the Hemoglobin Beta (HBB) protein using computational bioinformatics tools.

# Objective
To predict the three-dimensional structure of a protein and understand its structural features and functional regions.

#  Methodology
- Selected Hemoglobin Beta (HBB) protein sequence from UniProt
- Used SWISS-MODEL for 3D structure prediction
- Downloaded the predicted structure in PDB format
- Visualized and analyzed the structure using PyMOL

# Structural Analysis
- Identified key secondary structures: (Tool used: PyMol)
  - Alpha helices  
  - Beta sheets  
  - Loops and turns  
- Observed folding pattern and spatial arrangement of amino acids
- Analyzed regions important for oxygen binding and protein stability

# Results
- Successfully generated 3D model of HBB protein
- Structure shows characteristic globin fold
- Conserved structural regions indicate functional importance

# Files Included
- Protein structure file (.pdb)  
- Visualization images (screenshots)  
- Analysis report  

# Conclusion
The predicted structure of Hemoglobin Beta protein provides insights into its function and stability. Structural bioinformatics tools like SWISS-MODEL and PyMOL help in understanding protein architecture and biological roles.
